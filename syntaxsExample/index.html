Web Syntax Cheat Sheet

Switch between HTML and CSS references from the tabs below.

1. Basic Document

<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<title>Page Title</title>
	</head>
	<body>
		Content goes here.
	</body>
</html>

2. Headings and Text

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

<p>Paragraph text.</p>
<strong>Important text</strong>
<em>Emphasized text</em>
<mark>Highlighted text</mark>
<small>Small print</small>
<br>  <!-- line break -->
<hr>  <!-- thematic break -->

3. Links and Navigation

<a href="https://example.com" target="_blank" rel="noopener">Visit Site</a>
<a href="#section-id">Jump to section</a>
<nav>
	<a href="/">Home</a>
	<a href="/about">About</a>
</nav>

4. Images, Audio, Video

<img src="image.jpg" alt="Description" width="300" height="200">

<audio controls>
	<source src="audio.mp3" type="audio/mpeg">
	Your browser does not support audio.
</audio>

<video controls width="400">
	<source src="video.mp4" type="video/mp4">
	Your browser does not support video.
</video>

5. Form Structure

<form action="/submit" method="post" autocomplete="on">
	<label for="name">Name</label>
	<input id="name" name="name" type="text" required>

	<label for="email">Email</label>
	<input id="email" name="email" type="email" required>

	<label for="role">Role</label>
	<select id="role" name="role">
		<option value="dev">Developer</option>
		<option value="designer">Designer</option>
	</select>

	<button type="submit">Submit</button>
</form>

6. Semantic Layout and Metadata

<header>Top content</header>
<main>Main content</main>
<section>Themed section</section>
<article>Standalone block</article>
<aside>Side content</aside>
<footer>Bottom content</footer>

<meta name="description" content="Page summary">
<link rel="stylesheet" href="style.css">
<script src="app.js" defer></script>

1. Basic Rule Syntax

selector {
	property: value;
	property-two: value;
}

2. Selectors

* { box-sizing: border-box; }
body { margin: 0; }
.card { padding: 1rem; }
#main-title { color: #0f172a; }
nav a { text-decoration: none; }
ul > li { list-style: none; }
h2 + p { margin-top: 0; }
.item:hover { background: #eef2ff; }
input:focus { outline: 2px solid #0b6bcb; }
p::first-letter { font-size: 1.4rem; }

3. Colors and Backgrounds

:root {
	--brand: #0b6bcb;
	--text: #1f2937;
}

.badge {
	color: rgb(255, 255, 255);
	background: var(--brand);
	border: 1px solid hsl(215, 70%, 35%);
}

body {
	background: linear-gradient(180deg, #eef3ff, #f6f7fb);
}

4. Typography

body {
	font-family: "Segoe UI", Tahoma, sans-serif;
	font-size: 16px;
	font-weight: 400;
	line-height: 1.5;
	letter-spacing: 0.02em;
	text-align: left;
	text-transform: none;
	text-decoration: none;
}

5. Box Model

.box {
	width: 320px;
	height: 180px;
	margin: 1rem;
	padding: 1rem;
	border: 2px solid #d8deea;
	border-radius: 12px;
	box-sizing: border-box;
}

6. Display and Positioning

.inline { display: inline; }
.block { display: block; }
.none { display: none; }

.relative-parent { position: relative; }
.absolute-child {
	position: absolute;
	top: 10px;
	right: 10px;
}

.sticky-nav {
	position: sticky;
	top: 0;
}

7. Flexbox

.row {
	display: flex;
	flex-direction: row;
	justify-content: space-between;
	align-items: center;
	gap: 1rem;
	flex-wrap: wrap;
}

.item {
	flex: 1 1 200px;
}

8. Grid

.grid {
	display: grid;
	grid-template-columns: repeat(3, minmax(180px, 1fr));
	grid-auto-rows: minmax(120px, auto);
	gap: 1rem;
}

.feature {
	grid-column: 1 / 3;
}

9. Transitions and Animation

.button {
	transition: transform 180ms ease, background-color 180ms ease;
}

.button:hover {
	transform: translateY(-2px);
}

@keyframes fade-up {
	from { opacity: 0; transform: translateY(10px); }
	to { opacity: 1; transform: translateY(0); }
}

.panel {
	animation: fade-up 300ms ease;
}

10. Responsive Design

.container {
	width: min(960px, 100% - 2rem);
	margin-inline: auto;
}

@media (max-width: 700px) {
	.tabs {
		flex-direction: column;
	}

	.grid {
		grid-template-columns: 1fr;
	}
}

11. Common Shorthand

.card {
	margin: 1rem 0;
	padding: 1rem 1.25rem;
	border: 1px solid #d8deea;

	font: 500 1rem/1.4 "Segoe UI", sans-serif;
	transition: all 200ms ease;
}

13. Flip Card Demo

.flip-card {
	perspective: 1000px;
	width: 220px;
	height: 140px;
}

.flip-card-inner {
	position: relative;
	width: 100%;
	height: 100%;
	transform-style: preserve-3d;
	transition: transform 500ms ease;
}

.flip-card.is-flipped .flip-card-inner {
	transform: rotateY(180deg);
}

.flip-card-face {
	position: absolute;
	inset: 0;
	backface-visibility: hidden;
	border-radius: 12px;
	display: grid;
	place-items: center;
}

.flip-card-back {
	transform: rotateY(180deg);
}
